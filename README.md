# Personal Expense Tracker

A full-stack web application that helps users manage their personal finances by tracking income and expenses through a simple and intuitive interface.

## Features

- User authentication
- Add, edit, and delete income records
- Add, edit, and delete expense records
- Dashboard displaying financial summary
- Categorize transactions
- View transaction history
- Responsive and user-friendly interface

## Installation

### Clone the repository

```bash
git clone https://github.com/FahmidaFathima/Personal-Expense-Tracker.git
```

### Navigate to the project folder

```bash
cd Personal-Expense-Tracker
```

### Create a virtual environment

```bash
python -m venv .venv
```

### Activate the virtual environment

**Windows**

```bash
.venv\Scripts\activate
```

**Linux/macOS**

```bash
source .venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

The application will start locally.
